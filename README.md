# Scrapy Parser

## Тестовое задание

Используя фреймворк **Scrapy**, необходимо написать парсер для получения информации о товарах интернет-магазина из списка категорий по заранее заданному шаблону.
Информация должна быть структурирована в виде словарей согласно формату выходных данных и сохранена в файл формата **JSON**.
На вход подается список категорий в виде ссылок (минимум 3), с количеством товаров от 100 штук на категорию на сайте [alkoteka.com](https://alkoteka.com) (например: https://alkoteka.com/catalog/slaboalkogolnye-napitki-2)

## Описание
- Учитывается выбранный регион для парсинга — Краснодар.  
- Прокси не используется.

## Категории
Обрабатываются следующие категории:

- Слабоалкогольные напитки  [Ссылка](https://alkoteka.com/catalog/slaboalkogolnye-napitki-2)
- Безалкогольные напитки  [Ссылка](https://alkoteka.com/catalog/bezalkogolnye-napitki-1)
- Аксессуары  [Ссылка](https://alkoteka.com/catalog/aksessuary-2)
